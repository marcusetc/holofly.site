Política de Privacidade e Proteção de Dados PARA USUÁRIOS CRIATIVOS

Última atualização: 21 de janeiro de 2026

Esta política descreve como a Holofly Serviços de Tecnologia LTDA (CNPJ: 59.240.195/0001-90) coleta, utiliza e protege os dados fornecidos por você, Usuário Criativo, ao usar o aplicativo. Atuamos como intermediadores entre você, Usuário Criativo e os Usuários Clientes da plataforma.

1. Quais dados coletamos?

1.1. Dados fornecidos por você:

• Nome completo, e-mail, número de telefone e senha.
• CPF (utilizado para validação de identidade e repasses financeiros via PIX).
• Foto de perfil, imagem de fundo e 4 vídeos para composição do seu portfólio.
• Respostas a avaliações e formulários.

2. Como utilizamos seus dados?

• Exibir seu perfil para Usuários Clientes.
• Viabilizar repasses financeiros (utilizando o CPF como chave PIX).
• Disponibilizar acompanhamento do status dos repasses.
• Melhorar a plataforma e a experiência do usuário.

3. Como protegemos seus dados?

• Criptografia em trânsito (HTTPS/TLS).
• Acesso restrito aos dados sensíveis.
• Controles internos e atualizações de segurança.
• Armazenamos arquivos de mídia dos pedidos por 3 (três) dias após a conclusão do serviço.
• Após este período, os arquivos são excluídos permanentemente, exceto quando exigido por lei, ordens judiciais ou investigações em andamento.

4. Seus direitos

• Acessar, corrigir ou excluir seus dados.
• Solicitar limitação ou portabilidade dos dados.
• Revisar decisões automatizadas.
• Canal de atendimento: suporte@holofly.com.br

5. Uso do CPF e repasses financeiros

5.1. Finalidades do CPF
Utilizamos o CPF informado no cadastro para:
• Validação de identidade e prevenção de perfis duplicados.
• Identificação bancária nos repasses via PIX.

5.2. Processo de repasse
Os repasses são realizados pela Holofly fora do aplicativo, que serve apenas como painel informativo. Os pagamentos dos Usuários Clientes são processados pelo Mercado Pago.

5.3. Segurança
O CPF é protegido com medidas específicas para dados sensíveis, sendo usado exclusivamente para as finalidades acima.

6. Atualizações desta política

Você será notificado no app ou por e-mail caso esta política seja alterada.

Dúvidas: suporte@holofly.com.br
